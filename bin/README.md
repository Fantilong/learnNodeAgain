# 存放命令行相关的代码
在Linux系统下，把JS文件当做Shell脚本来运行，要通过在 #! 注释来指定单签脚本使用的解释器。例如：将该路径下的.JS文件当做Shell脚本运行，需要在头部指定解释器，添加注释（#! /user/bin/env node）即：该.js文件的解释器是 node
